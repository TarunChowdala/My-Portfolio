## ğŸ¨ Customization

### Update Personal Information

1. **Header Section** (`components/Header.jsx`):
   - Update name, title, and description
   - Modify contact information
   - Update resume link

2. **About Section** (`components/About.jsx`):
   - Add your bio and story

3. **Skills** (`components/Skills.jsx`):
   - Add/remove skills as needed

4. **Projects** (`components/Work.jsx`):
   - Add your portfolio projects
   - Update project images in `public/assets/`

### Theme Colors

Edit `tailwind.config.js` to customize the color scheme:

```javascript
theme: {
  extend: {
    colors: {
      // Your custom colors
    }
  }
}
```
